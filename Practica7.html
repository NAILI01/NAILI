ISP a utilizar

El ISP que se utilizará es GITHUB PAGES porque después de una larga investigación y comparación entre ISP, se pudo observar que este es totalmente gratuito, permitiendo crear un repositorio de manera sencilla, el proceso para dar de alta la página en HTML es fácil, además no pide número de cuenta para la opción “gratis”, también brinda varias opciones en caso de que se requieran más servicios o herramientas, es necesario pagar por ello, pero para efectos de práctica este ISP brinda una opción gratis para que la página web creada sea visible por otros usuarios, además no contiene anuncios o spam, en mi opinión este ISP fue el más apto para llevar a cabo la práctica y el proyecto.
